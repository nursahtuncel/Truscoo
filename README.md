# TrustCo Beginner HTML/CSS Boilerplate

## Overview

Welcome to the TrustCo Beginner HTML/CSS.

This project serves as a foundational template
for building the landing page inspired by the TrustCo design.

It provides a structured starting point for you.

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed on your machine:

- live-server vscode extension or equivalent CLI tool

## Project Structure

```
trustco-beginner-html-css-boilerplate/
│
├── public/                     # Static assets
├── index.html                  # Main HTML file
├── styles/                     # CSS stylesheets
│   ├── sections/               # Section-specific styles
│   ├── global/                 # Global styles (e.g., reset, variables, buttons, input)
│        ├── styles/
│        ├── reset.css          # Reset styles
│        ├── variables.css      # Custom variables
│
└── README.md              # This Project documentation

> along with other configuration files
```
